# Portfolio Tracker - Admin Portal

Admin portal for Mithra Gold Portfolio Tracker. Manage members, gold rates, and trades.

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server on port 5173.

## Features

- Dashboard with statistics and activity
- Gold rate management
- Member management
- Trade creation and management
- Desktop-optimized interface with sidebar navigation
